# Absentee App
## Run locally
Open this folder
<br>
Run:
<br>
    ```pip install tensorflow flask pillow numpy```
<br>
Then you can run.

## Run on docker
Open this folder
<br>
Run:
<br>
    ```docker build -t <name> .```
<br>
This builds the docker container with the Dockerfile and only the contents of this folder
